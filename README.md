### 三 目录说明

	src 源代码目录
	| 
	|---mods js模块文件
    |---*.html html文件
	|
	|---resources 资源文件夹
	|   |---fonts 图标字体
	|   |---images 图片
	|   |---sass sass文件（以“_”开头或结尾命名的scss文件表示会被其他scss文件引用）
	|   |---style sass编译后的css文件
	|
	build 编译目录
